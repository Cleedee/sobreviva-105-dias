# 📱 Guia de Controles Touch

Guia completo para jogar **Sobreviva 105 Dias** em dispositivos móveis.

---

## 📲 Como Jogar no Celular ou Tablet

### Passo 1: Abrir o Jogo
1. Abra o navegador (Chrome, Safari, Firefox)
2. Navegue até a pasta do jogo
3. Abra o arquivo `index.html`

### Passo 2: Iniciar
1. Toque no botão **"Iniciar Jogo"**
2. Os controles touch aparecerão automaticamente

---

## 🎮 Controles Detalhados

### 🕹️ Joystick Virtual (Movimentação)

**Localização:** Canto inferior esquerdo da tela

```
        ┌─────────────┐
        │             │
        │   (toque)   │
        │             │
        └─────────────┘
              ↑
         Joystick
```

**Como usar:**
- Toque e arraste dentro do círculo
- O personagem se move na direção do arrasto
- Quanto mais longe arrastar, mais rápido se move
- Solte para parar

**Exemplos:**
| Arrasto | Movimento |
|---------|-----------|
| ↑ Para cima | Personagem anda para cima |
| ↓ Para baixo | Personagem anda para baixo |
| ← Para esquerda | Personagem anda para esquerda |
| → Para direita | Personagem anda para direita |
| ↗ Diagonal | Movimento diagonal |

### ⚔️ Botões de Ação

**Localização:** Canto inferior direito da tela

#### Botão Atacar (Vermelho)
```
    ┌─────┐
    │ ⚔️  │
    │Atacar│
    └─────┘
```

- **Toque rápido:** Um ataque
- **Toque mantido:** Ataques contínuos (com cooldown)
- Use para derrotar inimigos (lobos, ursos)

#### Botão Ação (Verde)
```
    ┌─────┐
    │ 👆  │
    │ Ação │
    └─────┘
```

- **Toque rápido:** Executa interação
- Use para:
  - Cortar árvores (com machado)
  - Coletar frutas
  - Beber água
  - Falar com crianças
  - Abrir celas (com chave)

#### Botão Inventário (Azul)
```
    ┌─────┐
    │ 🎒  │
    │ Itens│
    └─────┘
```

- **Toque:** Abre/fecha o inventário
- Dentro do inventário, toque nos slots para selecionar itens

### ⏸️ Botões Auxiliares

    └─────┘

## 💡 Dicas de Jogo Touch

### Combate
1. Use o joystick para se aproximar do inimigo
2. Olhe na direção do inimigo (indicador branco aponta)
3. Toque no botão Atacar
4. Mantenha distância para evitar dano

### Exploração
1. Mova o joystick suavemente para explorar
2. Fique perto de objetos interativos
3. Use o botão Ação quando aparecer o prompt

### Coleta
1. Aproxime-se de árvores, arbustos ou pedras
2. Toque em Ação
3. Item será adicionado ao inventário

### Inventário
1. Toque no botão Inventário
2. Toque em um item para ver detalhes
3. Toque em "Usar" para equipar/consumir
4. Toque em "Fechar" para voltar ao jogo

## ⚠️ Solução de Problemas

### Controles não aparecem?
- Verifique se está usando um dispositivo touch
- Recarregue a página (pull to refresh)

### Joystick não responde?
- Toque e segure dentro do círculo
- Arraste para a direção desejada
- Não toque fora do círculo

### Botões muito pequenos?
- Em tablets, os botões podem parecer menores
- Zoom na página pode ajudar (pinch to zoom)

### Jogo lento?
- Feche outras abas do navegador
- Limpe o cache do navegador
- Use uma conexão estável

- O jogo detecta automaticamente o último input usado
