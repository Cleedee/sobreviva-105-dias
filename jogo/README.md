# 🌲 Sobreviva 105 Dias em uma Floresta

Um jogo de sobrevivência onde você precisa sobreviver por 105 dias em uma floresta misteriosa e resgatar 6 crianças capturadas.

## 📱 Controles

### Computador (Teclado)
| Tecla | Ação |
|-------|------|
| WASD / Setas | Mover |
| E | Interagir / Coletar |
| I | Abrir Inventário |
| Espaço | Atacar |
| ESC | Pausar |

### 📲 Dispositivos Móveis (Touch)

O jogo possui controles touch automáticos para smartphones e tablets:

#### Joystick Virtual (Esquerda)
- **Arraste** o joystick para se mover na direção desejada
- O personagem se move na direção que você arrastar

#### Botões de Ação (Direita)
- ⚔️ **Atacar** - Toque para atacar inimigos
- 👆 **Ação** - Toque para interagir com objetos e personagens
- 🎒 **Itens** - Toque para abrir o inventário

#### Botões Auxiliares
- ⏸️ **Pausar** - Toque para pausar o jogo

### Dicas Touch
- **Toque rápido** em Atacar/Ação = executar ação uma vez
- **Mantenha pressionado** em Atacar/Ação = manter ação ativa (ex: atacar continuamente)
- Os controles touch aparecem automaticamente quando o jogo inicia em dispositivos móveis

## 🎮 Objetivo

1. Sobreviver por **105 dias**
2. Encontrar e **resgatar 6 crianças** aprisionadas
3. Manter sua **vida, fome e sede** acima de zero

## 🌍 Mecânicas

### Status
- ❤️ **Vida** - Reduz ao receber dano ou ficar com fome/sede
- 🍖 **Fome** - Reduz com o tempo, coma para repor
- 💧 **Sede** - Reduz com o tempo, beba água para repor

### Dia/Noite
- O jogo possui ciclo de dia e noite
- À noite, inimigos ficam mais perigosos
- Na lua cheia, o Monstro Bugado caça!

### Itens
- 🪓 **Machado** - Para cortar árvores
- ⛏️ **Picareta** - Para minerar pedras
- 🍎 **Maçã/Frutas** - Para comer
- 🥩 **Carne** - Para assar no fogueira
- 🔑 **Chaves** - Para abrir as celas

## 🚀 Como Jogar

1. Abra `index.html` em um navegador
2. Clique em "Iniciar Jogo"
3. Use WASD para se mover e explore o mundo!
4. Encontre e resgate todas as 6 crianças
5. Sobreviva por 105 dias para vencer!
